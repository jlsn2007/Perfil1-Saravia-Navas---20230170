¿Qué son los componentes pasivos en la Red?

Los componentes pasivos son aquellos elementos que no requieren energía eléctrica para funcionar. Estos componentes forman la estructura física de la red.

Componentes Pasivos

RJ45

El conector RJ45 es un conector de ocho pines que se usa comúnmente en redes Ethernet para conectar dispositivos a través de cables UTP.

Jacks o Keystone

Los Jacks o Keystone son conectores modulares utilizados en instalaciones de red para conectar cables a dispositivos como paneles de parcheo.

Cable UTP

El cable UTP (Unshielded Twisted Pair) es un tipo de cable de red que se utiliza para conexiones Ethernet. Consta de pares de cables trenzados sin blindaje.

Patch Panel

El Patch Panel es un dispositivo de red que organiza y conecta cables de red, facilitando la gestión y el mantenimiento de la red.

Patch Cord

El Patch Cord es un cable corto utilizado para conectar dispositivos en una red, como entre un Patch Panel y un switch.

Fibra Óptica

La fibra óptica es un medio de transmisión que utiliza luz para enviar datos a alta velocidad y largas distancias, comúnmente usada en redes troncales.

Canaletas

Las canaletas son conductos usados para organizar y proteger los cables en instalaciones de red, manteniendo un entorno ordenado y seguro.

¿Qué son los componentes activos en la Red?

Los componentes activos son dispositivos que requieren energía eléctrica para funcionar y desempeñan un papel crucial en la gestión y transmisión de datos en la red.

Componentes Activos

Switch

Un switch es un dispositivo de red que conecta varios dispositivos en una red, permitiendo la transmisión de datos entre ellos.

Router

Un router es un dispositivo de red que dirige el tráfico de datos entre diferentes redes, como entre una red local y una red externa como Internet.

Tarjeta de Red

La tarjeta de red es un componente de hardware que permite a una computadora conectarse a una red, facilitando la comunicación con otros dispositivos.

Interruptor Inteligente Wifi

Un interruptor inteligente Wifi es un dispositivo que permite controlar de manera remota la conexión de corriente a otros dispositivos a través de una red inalámbrica.

Access Point

Un Access Point es un dispositivo que permite a otros dispositivos conectarse a una red de área local inalámbrica, extendiendo la cobertura de la red.

Herramientas para Cableado Estructurado

Ponchadora

La ponchadora es una herramienta utilizada para insertar los cables en los conectores de red, asegurando un contacto firme y estable.

Crimpadora

La crimpadora es una herramienta que se utiliza para unir conectores a los cables de red mediante la aplicación de presión.

Desforradora

La desforradora es una herramienta usada para remover la cubierta exterior de los cables de red, dejando expuestos los conductores internos para su conexión.

Tester/Probador de Cables

El tester o probador de cables es una herramienta utilizada para verificar la continuidad y correcta conexión de los cables de red.

¿Qué es el cableado horizontal?

El cableado horizontal es la parte del sistema de cableado estructurado que se extiende desde los paneles de parcheo hasta las tomas de corriente en las áreas de trabajo.

¿Qué es el cableado backbone?

El cableado vertical o backbone es el sistema de cableado que conecta los diferentes pisos de un edificio o áreas de red a través de un punto central, como un centro de datos.

Tipos de Cables para Cableado Horizontal

Cable CAT5e

El cable CAT5e es una versión mejorada del CAT5, ofreciendo velocidades de hasta 1 Gbps y se utiliza en la mayoría de las instalaciones de red actuales.

Cable CAT6

El cable CAT6 permite velocidades de hasta 10 Gbps en distancias cortas y es común en redes que requieren mayor ancho de banda y velocidad.

Normativa para hacer cables de red (clase A y B)

Existen dos estándares principales para la confección de cables de red, conocidos como T568A y T568B. Estos definen el orden de los colores de los hilos dentro del conector RJ45.

T568A

El estándar T568A es comúnmente utilizado en aplicaciones residenciales y es compatible con ambas normas de cableado (A y B).

T568B

El estándar T568B es el más utilizado en instalaciones comerciales y define un orden de colores diferente al T568A para los cables de red.

Tipos de Redes

LAN (Local Area Network)

Una LAN es una red que conecta dispositivos en una área geográfica pequeña, como una oficina o un hogar.

MAN (Metropolitan Area Network)

Una MAN es una red que conecta dispositivos en una área geográfica más grande que una LAN, como una ciudad.

WAN (Wide Area Network)

Una WAN es una red que cubre un área geográfica extensa, conectando múltiples LANs y MANs, como la red global de Internet.
